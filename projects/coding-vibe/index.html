🧠 Layer 2: Orchestrator — OpenOPC Patterns + lopi Agent Pool 第二层：编排器——OpenOPC 模式 + lopi Agent 池

A lightweight daemon combining proven patterns from OpenOPC (HKUDS) with lopi's scored agent pool: 轻量守护进程，融合 OpenOPC（HKUDS）的成熟模式与 lopi 的评分 Agent 池：

  • HeartbeatScheduler (OpenOPC) — wakes agents every 30s to check pending tasks. In-process asyncio loop, no separate service needed. HeartbeatScheduler（OpenOPC）——每 30s 唤醒 agent 检查待办。进程内 asyncio 循环，无需独立服务。
  • EscalationEngine (OpenOPC) — blocks runtime, escalates key decisions to you via voice. Types: INFO_NEEDED, DECISION_NEEDED, RISK_WARNING. Timeout + default action. EscalationEngine（OpenOPC）——挂起运行，将关键决策通过语音上报给你。类型：信息需求、决策需求、风险警告。超时+默认动作。
  • Gate Harness (OpenOPC) — LLM judge evaluates completed work: pass / rework / escalate. Same blocker N times → auto-escalate to human. Gate Harness（OpenOPC）——LLM 裁判评估完成质量：通过/返工/上报。同一问题 N 次→自动上报给你。

  • Work-Item DAG (OpenOPC) — dependency graph of tasks. Independent items run in parallel; dependent items wait. Phase transitions: TODO → IN_PROGRESS → IN_REVIEW → DONE. Work-Item DAG（OpenOPC）——任务依赖图。独立项并行执行；依赖项等待前序完成。状态流转：TODO→IN_PROGRESS→IN_REVIEW→DONE。

WHAT MAKES IT DIFFERENT为什么不同

⚡ Why Different核心差异

1. Always-Running, Not a Chat Window永远在线，不是聊天窗
A daemon that runs 24/7, like a real team. Current tools are chat windows you open and close. 一个 24/7 运行的守护进程，像一个真正的团队。现有工具都是你打开再关闭的聊天窗。
2. Multi-Agent, Not One LLM多 Agent 协作，不是单模型
Specialized agents with bounded contexts, debating and reviewing each other. Not one model trying to do everything. 专业 Agent 各司其职，互相辩论和审查。不是一个模型试图做所有事。
3. Decision Gate, Not Command Interface决策门，不是指令接口
You don't tell it what to do step by step. It tells you what decisions it needs. You're the CEO, not the operator. 你不用一步步告诉它做什么。它告诉你需要做什么决策。你是 CEO，不是操作员。
4. Hardware-Native, Not a Web App硬件原生，不是 web app
Designed for earphones + ring + glasses. You interact while walking, at the gym, in bed. 为耳机+戒指+眼镜而设计。你走路、健身、躺床上时都能交互。
5. Full Pipeline, Not Just Coding全流程，不只是写代码
Code → Review → Deploy → Market — all autonomous. Not a coding tool. A company factory. 编码→审查→部署→宣发——全部自主执行。不是编程工具。是公司工厂。
