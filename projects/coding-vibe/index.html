🧠 Layer 2: Orchestrator (Always-Running Daemon) 第二层：编排器（常驻守护进程）

The brain. A persistent process with three core components: 大脑。一个永不停止的进程，包含三个核心组件：

  • Watcher Agent — listens to hardware stream, filters signal from noise 监听 Agent——感知硬件输入，过滤噪音，识别意图
  • Router Agent — decides: dispatch to agent pool? spawn new agent? escalate to user? 路由 Agent——决策：分派到 Agent 池？创建新 Agent？还是上报给你？
  • Decision Gate — ONLY interrupts you for high-stakes decisions. Everything else: autonomous. 决策门——仅在高风险决策时打断你。其余全部自主执行。
⚙️ Layer 3: Claude Code Agent Pool 第三层：Claude Code Agent 池

Specialized agents running in parallel git worktrees. Spawned and killed by the orchestrator as needed. 多个专业 Agent 并行运行在独立 git worktree 中。由编排器按需创建和回收。

WHAT MAKES IT DIFFERENT为什么不同

⚡ Why Different核心差异

1. Always-Running, Not a Chat Window永远在线，不是聊天窗
A daemon that runs 24/7, like a real team. Current tools are chat windows you open and close. 一个 24/7 运行的守护进程，像一个真正的团队。现有工具都是你打开再关闭的聊天窗。
2. Multi-Agent, Not One LLM多 Agent 协作，不是单模型
Specialized agents with bounded contexts, debating and reviewing each other. Not one model trying to do everything. 专业 Agent 各司其职，互相辩论和审查。不是一个模型试图做所有事。
3. Decision Gate, Not Command Interface决策门，不是指令接口
You don't tell it what to do step by step. It tells you what decisions it needs. You're the CEO, not the operator. 你不用一步步告诉它做什么。它告诉你需要做什么决策。你是 CEO，不是操作员。
4. Hardware-Native, Not a Web App硬件原生，不是 web app
Designed for earphones + ring + glasses. You interact while walking, at the gym, in bed. 为耳机+戒指+眼镜而设计。你走路、健身、躺床上时都能交互。
5. Full Pipeline, Not Just Coding全流程，不只是写代码
Code → Review → Deploy → Market — all autonomous. Not a coding tool. A company factory. 编码→审查→部署→宣发——全部自主执行。不是编程工具。是公司工厂。
